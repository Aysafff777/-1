# Random Quote Generator

## Автор: Шарифуллин Айсаф Эдуардович
## Дата: 04.05.2026
## Вариант: Random Quote Generator


## Описание
Приложение для генерации случайных цитат с фильтрацией по автору и теме, сохранением истории в JSON.

## Примеры использования
1. Запустите main.py.
2. Нажмите «Сгенерировать цитату».
3. Используйте фильтрацию по автору/теме.
4. История сохраняется автоматически.
